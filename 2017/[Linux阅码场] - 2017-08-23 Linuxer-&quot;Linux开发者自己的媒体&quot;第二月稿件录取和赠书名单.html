写点啥呢？

去互联网邮局，寄个“包裹”（分享你对一个问题的见解）给我们；但表关心国家大事。

包裹类型：

1.高屋建瓴的谈技术

讨论Linux相关的任何技术话题，篇幅不限，要求能比较深刻到位地说清楚一个问题。


2.务实地讨论学习方法、职业规划、人生感悟的文章

讨论自己是如何走向Linux，从事该职业方向，以及工作感悟的文章；讨论自己学习和开发Linux心得的文章；大牛指导新人少走弯路的文章。


3.认真地提出一个有价值的问题

你可以认真的提出一个你学习和工作中碰到的问题，一旦我们觉得这是个好问题，会有大咖来帮你说清楚这个问题。


4.天马行空地会来事

我们鼓励任何天马行空，能落地的想法，你如果有，告诉我们。学习和工作中有困难，除了找“man”，还可以找linuxer。
